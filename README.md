# GP-To-Do-List-App-
Practice basic software engineering workflow using a very small application.  This assignment is not about features or polish. It is about:  Working in a shared repository. Planning before coding. Communicating progress like a real team. This To-List App should have features such as:  Add task Remove task Mark Task as complete Due dates etc. 

## What the MVP does
- Allows adding tasks with due date.
- Displays all tasks with their status. 
- Marks tasks as completed.
- Remove tasks.
- Runs in the console and allows the user to exit the program.
